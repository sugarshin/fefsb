# fefsb

*Front-end Full Stack Boilerplate*

- React
- Redux
- RxJS (redux-observable)
- webpack 2
- Babel
- CSS Modules
- ESLint
- Flow
- PostCSS

## Getting started

```sh
yarn
```

## Usage

```sh
# Start dev server
yarn start

# Build
yarn build
```
